CSS3 animation-direction 属性


实例

animation-direction:alternate;
-webkit-animation-direction:alternate; /* Safari and Chrome */

尝试一下 ?

浏览器支持

Internet Explorer 10, Firefox, 和 Opera支持animation-direction 属性.

Safari和Chrome通过私有属性-WebKit-animation-direction支持。

注意: Internet Explorer 9及更早IE版本不支持animation-direction 属性。

标签定义及使用说明

animation-direction属性定义动画是否应该播放完后逆向交替循环。

如果动画方向的值是"alternate"，动画正向播放奇数次迭代，并反向播放偶数次迭代。

Note: 如果动画是设定运动只有一次，这个属性不会有任何效果。

默认值: normal
继承: no
版本: CSS3
JavaScript 语法: object object.style.animationDirection="alternate"

语法

animation-direction: value;

值 描述
normal 默认值。动画应该运动正常 测试 ?
reverse 动画应该轮流反向播放。
alternate 如果动画方向的值是"alternate"，动画正向播放奇数次迭代(1,3,5等) ，并反向播放偶数次迭代(2,4,6等)。
alternate-reverse 动画反向播放奇数次迭代(1,3,5等) ，并正向播放偶数次迭代(2,4,6等)。
