CSS clear 属性


实例

指定段落的左侧或右侧不允许浮动的元素：

img
{
float:left;
}
p.clear
{
clear:both;
}

尝试一下 »
本页底部查看更多实例。

属性定义及使用说明

clear属性指定段落的左侧或右侧不允许浮动的元素。

默认值： none
继承性： no
版本： CSS1
JavaScript 语法： object.style.clear="left"

浏览器支持

所有主流浏览器都支持clear属性。

注意:IE7和更早的版本不支持"inherit"的值。IE8需要定义！DOCTYPE。IE9支持 "inherit".

属性值

值 描述
left 在左侧不允许浮动元素。
right 在右侧不允许浮动元素。
both 在左右两侧均不允许浮动元素。
none 默认值。允许浮动元素出现在两侧。
inherit 规定应该从父元素继承 clear 属性的值。
