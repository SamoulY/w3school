XML DOM 验证 XML 语法

XML 错误将会停止您的 XML 程序

XML 文档中的错误将停止您的 XML 程序。

W3C XML 规范规定如果发现错误，程序不应该继续处理 XML 文档。究其原因是 XML 软件应该是易于编写的，所有的 XML 文档应该是兼容的。

通过 HTML 就有可能创建带有"错误"（如缺少结束标签）的文档。HTML 浏览器如此巨大且不相容的主要原因之一是在遇到 HTML 错误时它们都有自己的方法弄明白文档本应该是什么样的。

通过 XML，这应该是不可能的。
