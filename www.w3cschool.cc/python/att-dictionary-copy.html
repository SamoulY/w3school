Python 字典(Dictionary) copy()方法

描述

Python 字典(Dictionary) copy() 函数返回一个字典的浅复制。

语法

copy()方法语法：

dict.copy()

参数

  • NA。

返回值

返回一个字典的浅复制。

实例

以下实例展示了 copy()函数的使用方法：

#!/usr/bin/python

dict1 = {'Name': 'Zara', 'Age': 7};

dict2 = dict1.copy()
print "New Dictinary : %s" %  str(dict2)

以上实例输出结果为：

New Dictinary : {'Age': 7, 'Name': 'Zara'}
