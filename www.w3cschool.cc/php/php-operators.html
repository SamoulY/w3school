PHP 赋值运算符

在 PHP 中，基本的赋值运算符是 "="。它意味着左操作数被设置为右侧表达式的值。也就是说，"$x = 5" 的值是 5。

运算符 等同于 描述
x = y x = y 左操作数被设置为右侧表达式的值
x += y x = x + y 加
x -= y x = x - y 减
x *= y x = x * y 乘
x /= y x = x / y 除
x %= y x = x % y 模（除法的余数）
a .= b a = a . b 连接两个字符串

PHP 比较运算符

比较操作符可以让您比较两个值：

运算符 名称 描述 实例
x == y 等于 如果 x 等于 y，则返回 true 5==8 returns false
x === y 恒等于 如果 x 等于 y，且它们类型相同，则返回 true 5==="5" returns false
x != y 不等于 如果 x 不等于 y，则返回 true 5!=8 returns true
x <> y 不等于 如果 x 不等于 y，则返回 true 5<>8 returns true
x !== y 不恒等于 如果 x 不等于 y，且它们类型不相同，则返回 true 5!=="5" returns true
x > y 大于 如果 x 大于 y，则返回 true 5>8 returns false
x < y 小于 如果 x 小于 y，则返回 true 5<8 returns true
x >= y 大于等于 如果 x 大于或者等于 y，则返回 true 5>=8 returns false
x <= y 小于等于 如果 x 小于或者等于 y，则返回 true 5<=8 returns true
