HTML DOM Style 对象

Style 对象

Style 对象代表单个样式的声明。

Style 对象可从应用样式的文档或元素访问。

使用 Style 对象属性的语法：

document.getElementById("id").style.property="value"

Style 对象的属性类别：

  • Background
  • Border/Outline
  • Generated Content
  • List
  • Misc
  • Margin/Padding
  • Positioning/Layout
  • Printing
  • Table
  • Text
