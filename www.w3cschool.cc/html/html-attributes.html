HTML 属性

属性是 HTML 元素提供的附加信息。

HTML 属性

  • HTML 元素可以设置属性
  • 属性可以在元素中添加附加信息 about an element
  • 属性一般描述于开始标签
  • 属性总是以名称/值对的形式出现，比如：name="value"。

属性实例

HTML 链接由 <a> 标签定义。链接的地址在 href 属性中指定：

实例

<a href="http://www.w3cschool.cc">This is a link</a>

HTML 提示：使用小写属性

属性和属性值对大小写不敏感。

不过，万维网联盟在其 HTML 4 推荐标准中推荐小写的属性/属性值。

而新版本的 (X)HTML 要求使用小写属性。
